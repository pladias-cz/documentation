History

  • project started on 2014, first launch 2016 - instance for vascular plants of the Czech Republic Pladias
  • 2019 new instance for nonvascular plants of the Czech Republic DaLiBor
  • 2021 new instance for vegetation traits (having no occurrence data included) CEVS
  • 2024 new instance for vascular plants of the Slovakia Pladias.sk

Technology stack

  • PostgreSQL + PostGIS + PostgREST
  • Java with Play! Framework
  • Geoserver
